# app

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.7. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

# 前端项目

1. 创建

```bash
bun create vite@latest
```

2. 安装依赖

```bash
bun install
```

3. 更新脚本

```
  "dev": "bunx --bun vite"
```
